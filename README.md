# Manipulação e Análise de Dados com Python

Projeto desenvolvido para estudo de manipulação de dados utilizando Python, Pandas e requisições HTTP.

## Objetivo

Realizar:
- Coleta de dados via requisições HTTP
- Tratamento e limpeza de dados
- Análise exploratória
- Organização de código em módulos 

## Tecnologias Utilizadas

- Python 3.x
- Pandas
- Requests
- LXML
- Jupyter Notebook
